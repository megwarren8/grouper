# Grouper: a random student grouper for the classroom

Paste (or import) a class list, pick a group size, and press **Group**. Grouper shuffles the names into balanced groups of the size you asked for, side by side on one screen so a class can walk in, glance at the projector, and go sit with their group. Offline, no accounts, no data leaves the page.

Designed and built by [Megan Warren](https://github.com/megwarren8).

## What makes it useful

- **One screen, no setup.** Type or paste a roster, spin, project. That is the whole loop.
- **Balanced groups.** Group size is respected exactly, with the remainder distributed so no group is left with just one student.
- **Reshuffle without retyping.** The roster persists on the page. Groupings do not, on purpose, so the next spin is genuinely random.
- **Absent students without editing the roster.** A tap marks a name absent; it sits out this round without being deleted.
- **Reads well on a projector.** Large type, high contrast, clean layout.

## Run it

Open `index.html` in any modern browser. No install, no server, no build step.

For a Mac desktop shortcut, `start.command` opens `index.html` in the default browser on double-click.
